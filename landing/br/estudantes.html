Cafezin para Estudantes e Pesquisadores

Anotações de pesquisa, capítulos de TCC e artigos — tudo em arquivos Markdown simples que são totalmente seus. O Cafezin adiciona IA para rascunhar e revisar, um canvas para estruturar argumentos complexos e controle de versão via git para que cada rascunho seja recuperável.

Atualizado: março de 2026

Por que importa: A escrita acadêmica é longa e iterativa. Arquivos guardados em apps proprietários desaparecem quando a assinatura expira. O Cafezin mantém tudo em Markdown puro, localmente, com histórico completo de versões.

Sua tese, seus arquivos, seu controle

Todo documento no Cafezin é um arquivo .md no seu disco. Sem lock-in, sem dependência de nuvem, sem risco de perder acesso quando uma assinatura vence. Versione cada rascunho com git integrado — exatamente como desenvolvedores controlam código.

IA que mostra o que mudou

Peça à IA do Cafezin para expandir uma seção, afiar um argumento ou simplificar linguagem técnica para um resumo. Cada mudança feita pela IA é rastreada e revisável — você decide o que entra e o que não entra. Sem reescritas invisíveis.

Canvas para estruturar ideias complexas

Antes de escrever um capítulo, mapeie-o no canvas. Mova seções, desenhe relações entre ideias, construa esboços visuais. O canvas fica ao lado dos seus documentos no mesmo workspace.

Capture ideias em qualquer lugar com voz

Teve um insight enquanto lia? Toque para gravar e fale. O Cafezin transcreve e salva como nota que você pode puxar para qualquer documento depois.

Funciona completamente offline

Biblioteca, campo, avião — o Cafezin funciona onde você estiver, com ou sem internet. Seus arquivos e sua IA são locais depois da configuração.

Perguntas rápidas

Estudantes perguntam

O Cafezin exporta para Word ou PDF?

O Cafezin salva em Markdown puro. Converta para Word ou PDF com Pandoc, Typora ou qualquer conversor Markdown padrão — os arquivos são sempre seus e portáveis.

Posso usar para notas de múltiplas disciplinas?

Sim. Crie um workspace por disciplina ou semestre. Os arquivos ficam no seu Mac, organizados como preferir. O Cafezin não impõe estrutura de pastas.

A IA serve para escrita acadêmica?

Sim — para rascunhar, reescrever seções densas, gerar esboços e simplificar linguagem para resumos. A IA mostra o que mudou para você revisar antes de aceitar.
